# Laravel Social App APIs

## Clone the repo
```
git clone https://github.com/CodeWithHBKY/laravel-social-apis.git
```

## After installation

### Step 1
```
cd laravel-social-apis/
```

### Step 2
```
composer update
```
or

```
composer install
```

## Run project

```
php artisan server
```

### Migrate

```
php artisan migrate
```
